# @fesjs/plugin-windicss

即将废弃

由于 windicss 不怎么维护了，本插件即将废弃，推荐使用 tailwindcss在新窗口打开。

# 介绍

windicss 支持

# 启用方式

在 package.json 中引入依赖：

{
    "devDependencies": {
        "@fesjs/plugin-windicss": "^2.0.0"
    }
}

# 配置

在 .fes.js 配置文件中添加自定义配置，详细配置请看在新窗口打开：

export default {
    windicss: {
        root: './',
    },
};
在 GitHub 上编辑此页 在新窗口打开
